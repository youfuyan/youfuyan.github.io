## Recognition

Equal-contribution work published in IEEE Transactions on Visualization and
Computer Graphics and recognized with an IEEE VIS 2024 Honorable Mention.

## Research question

How can an interface help people inspect, connect, and understand health information
returned by an LLM instead of accepting a fluent answer at face value?

## System

KNOWNet combines:

- a conversational interface
- retrieval-augmented generation
- knowledge-graph validation
- progressive graph visualization
- recommendation support for follow-up exploration

## Contribution

Youfu worked across system design and implementation using OpenAI GPT-4, Flask,
Neo4j, Next.js, and D3.js.

## Result

The system improved user comprehension by 20% in the reported evaluation.

## Design lesson

Trustworthy AI is not solved by adding a disclaimer. Interfaces can expose
relationships, evidence, and exploration paths that help users reason about an answer.
